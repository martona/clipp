# Clipp v1.0

Secure cross-platform clipboard sync for trusted devices.

Clipp is a free, open source, peer-to-peer LAN-only clipboard sync utility that currently supports Windows, macOS and iOS.

I desperately needed the functionality, and while there are tons of solutions that let you do the same thing, they check one or more of the following boxes, which made them a non-starter: not free, not open source, cloud-based, or kitchen-sink type apps.

If you need WAN-syncing, Tailscale or NostrVPN work great with Clipp.
